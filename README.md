# Simple Calculator

A basic calculator interface built with HTML, CSS, and JavaScript. The project supports arithmetic operations, interactive buttons, and percentage calculations.

## Current Status

### Functional Features
- **HTML structure** – Buttons, display, and layout are fully implemented.
- **CSS styling** – Clean, responsive design.
- **JavaScript functionality** – Supports:
  - Keyboard input support
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Percentage (%) – Calculates percentages of the current value
  - Backspace button – Removes the last character or operator
  - Clear All (C) button – Resets the display
  - Equals (=) button – Calculates the result
  - Decimal button – Fully functional

### Remaining Features
- Maintain a history of calculations

## Usage
1. Click numbers and operators to perform calculations.
2. Click **Backspace** to delete the last input.
3. Click **C (Clear All)** to reset the calculator.
4. Press **=** to calculate the result.
5. Use the **%** button to calculate percentages.
6. Decimal points work as expected.
7. Keyboard input is supported.
